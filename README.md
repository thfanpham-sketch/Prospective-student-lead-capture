# Prospective-student-lead-capture
MIS Project: Lead capture and analytics for prospective students

## Problem
No online form exists; Marketing cannot proactively nurture leads; Contact info is inconsistent.

## Solution
- Created lead capture form requirements
- Designed data flow into CRM (Salesforce)
- Developed process flow diagram
- Built Power BI dashboard for key metrics

## Key Metrics
- Leads per course, opt-in rate, conversion funnel

## Tools
Power BI, Excel, Draw.io, Word
